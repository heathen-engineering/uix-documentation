Detailed Description

Moves each transform in the corasponding relative direction at the corasponding speed

Directions should contain a local space direction vector for each transform e.g. 0,0,1 is the same as the transform.forward would be in mono behaviours Speeds should contain a speed expressed as units per second for each transform
