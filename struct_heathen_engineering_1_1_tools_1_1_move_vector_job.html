Detailed Description

Updates a set of vectors to move in coraponding directions by corasponding speeds

Vectors should be a list of the current positions and will be updated to represent the new positions Directions should be a 1 for 1 list of world space directions Speeds should be a 1 for 1 list of unit speed per second
